# OG Image 说明

## 需要创建的图片

### og-image.png
- **尺寸**: 1200 x 630 像素
- **用途**: 社交媒体分享预览图
- **内容建议**:
  - PictriKit Logo
  - 标语: "Free Online Screenshot Stitching Tool"
  - 简洁的工具界面截图
  - 品牌色: #0066cc

### 设计要点
1. 清晰的品牌标识
2. 简洁的文字说明
3. 高对比度，易于识别
4. 适合在小尺寸下显示

### 放置位置
将 `og-image.png` 放在 `assets/images/` 目录下
