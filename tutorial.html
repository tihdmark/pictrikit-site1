How to Use ScreenStitch

A complete guide to combining screenshots like a pro

Getting Started

ScreenStitch is a powerful yet simple tool for combining multiple screenshots into a single image. Whether you're creating tutorials, documenting bugs, or sharing conversations, this guide will help you master all the features in just a few minutes.

No installation required! Simply open ScreenStitch in your web browser and start combining screenshots immediately. All processing happens locally on your device, ensuring your images remain private and secure.

Step 1: Upload Your Screenshots

There are three easy ways to add screenshots to ScreenStitch:

Method 1: Click to Upload

  1. Click the "Upload" button (folder icon) in the left sidebar
  2. Select one or multiple screenshot files from your computer
  3. Supported formats: PNG, JPG, JPEG, GIF, WebP
  4. Your images will appear on the canvas immediately

Method 2: Drag and Drop

  1. Open your file explorer or finder
  2. Select the screenshots you want to combine
  3. Drag them directly onto the ScreenStitch canvas
  4. Release to add them to your workspace

Method 3: Paste from Clipboard

  1. Take a screenshot using your system's screenshot tool (e.g., Windows: Win+Shift+S, Mac: Cmd+Shift+4)
  2. Click on the ScreenStitch canvas
  3. Press Ctrl+V (Windows) or Cmd+V (Mac)
  4. The screenshot will be added instantly

Pro Tip: You can mix and match these methods! Upload some images, then paste others from your clipboard.

Step 2: Arrange Your Screenshots

Once your screenshots are uploaded, it's time to arrange them into the perfect layout.

Moving Screenshots

  • Click and drag any screenshot to move it around the canvas
  • Position screenshots vertically, horizontally, or in any custom layout
  • Overlap screenshots if needed for creative effects

Smart Alignment (Recommended)

The Smart Align feature (magnet icon) is enabled by default and helps you align screenshots perfectly:

  • Edge Snapping: Screenshots automatically snap to each other's edges when you drag them close
  • Alignment Guides: Visual guides appear to show when screenshots are aligned
  • Precise Positioning: Ensures pixel-perfect alignment without manual adjustment
  • Toggle On/Off: Click the magnet icon to enable or disable smart alignment

Resizing Screenshots

  • Click on a screenshot to select it (you'll see corner handles)
  • Drag the corner handles to resize proportionally
  • Hold Shift while dragging to maintain aspect ratio
  • Drag edge handles to stretch in one direction

Rotating Screenshots

  • Select a screenshot by clicking on it
  • Hover near a corner until you see the rotation cursor
  • Click and drag to rotate freely
  • The rotation angle is displayed while rotating

Step 3: Use Editing Tools

ScreenStitch includes powerful editing tools to enhance your combined screenshots.

Drawing Tool

Add annotations, highlights, or freehand drawings:

  1. Click the Draw button (paintbrush icon) in the sidebar
  2. Choose your brush color from the right panel
  3. Adjust the brush size using the slider (1-50 pixels)
  4. Click and drag on the canvas to draw
  5. Click the Draw button again to exit drawing mode

Use Cases: Circle important areas, draw arrows, add emphasis, or create custom annotations.

Text Tool

Add text labels, captions, or explanations:

  1. Click the Add Text button (font icon)
  2. A text box will appear on the canvas
  3. Double-click the text to edit it
  4. Customize in the right panel:
    • Font family (Arial, Helvetica, Georgia, etc.)
    • Font weight (Normal or Bold)
    • Text color
  5. Move and resize the text box like any other object

Crop Tool

Trim images to focus on important areas:

  1. Select an image on the canvas
  2. Click the Crop button (crop icon) in the sidebar
  3. A crop rectangle will appear with semi-transparent overlays showing areas to be removed
  4. Drag the crop rectangle to reposition it
  5. Drag the corner handles to resize the crop area
  6. Click Confirm to apply the crop, or Cancel to exit without changes

Keyboard Shortcuts: Press K to toggle crop mode, Enter to confirm, Escape to cancel.

Tip: You can crop the same image multiple times to refine your selection. Use Ctrl+Z to undo if needed.

Flip Tools

  • Flip Horizontal: Mirror the selected screenshot left-to-right
  • Flip Vertical: Mirror the selected screenshot top-to-bottom
  • Useful for correcting orientation or creating mirror effects

Step 4: Customize the Background

Choose the perfect background for your combined screenshots using the right panel.

Transparent Background

  • Click the "Transparent" option
  • Perfect for overlaying on other images or presentations
  • Exports as PNG with transparency preserved

Solid Color Background

  • Click the "Color" option
  • Choose any color using the color picker
  • Great for creating consistent branding or matching themes

Image Background

  • Click the "Image" option to upload a background image
  • Or use the "Set Image as Canvas Background" button to use one of your uploaded screenshots as the background
  • Useful for creating composite images or themed layouts

Step 5: Zoom and Navigate

Work comfortably with large canvases using zoom and pan controls.

Zoom Controls

  • Zoom In: Click the + button (or use mouse wheel up)
  • Zoom Out: Click the - button (or use mouse wheel down)
  • Reset Zoom: Click the expand button to fit the canvas to your screen
  • Current zoom level is displayed between the buttons

Pan the Canvas

  • Hold Spacebar and drag to pan around the canvas
  • Useful when zoomed in to navigate large compositions
  • Release spacebar to return to normal editing mode

Step 6: Undo and Redo

Made a mistake? No problem!

  • Undo: Click the undo button (or press Ctrl+Z / Cmd+Z)
  • Redo: Click the redo button (or press Ctrl+Y / Cmd+Y)
  • ScreenStitch remembers your entire editing history
  • Experiment freely knowing you can always go back

Step 7: Delete and Clear

Delete Selected Object

  • Select the screenshot, text, or drawing you want to remove
  • Click the Delete button (trash icon)
  • Or press the Delete or Backspace key

Clear Everything

  • Click the Clear button (broom icon) to remove all objects
  • Useful for starting fresh with a new project
  • You'll be asked to confirm before clearing

Step 8: Export Your Combined Screenshot

Once you're happy with your combined screenshot, it's time to save it.

Download Options

  1. Click the Download button (download icon) in the top toolbar
  2. Your combined screenshot will be saved to your default downloads folder
  3. File format: PNG (high quality with transparency support)
  4. File name: Automatically generated with timestamp

Share Options

  1. Click the Share button (share icon)
  2. Choose from available options:
    • Copy to Clipboard: Paste directly into other applications
    • Download: Save to your device

Pro Tip: After copying to clipboard, you can paste directly into email, chat apps, documents, or image editors.

Advanced Tips and Tricks

Keyboard Shortcuts

  • Ctrl/Cmd + V: Paste screenshot from clipboard
  • Ctrl/Cmd + Z: Undo last action
  • Ctrl/Cmd + Y: Redo last undone action
  • Delete/Backspace: Delete selected object
  • Spacebar + Drag: Pan the canvas
  • Mouse Wheel: Zoom in/out

Best Practices

  • Plan Your Layout: Think about the final arrangement before uploading
  • Use Smart Align: Keep it enabled for professional-looking results
  • Consistent Sizing: Resize screenshots to similar dimensions for better visual flow
  • Add Context: Use text and drawings to explain what viewers should focus on
  • Choose Backgrounds Wisely: Transparent for flexibility, solid colors for clarity
  • Save Frequently: Download intermediate versions if working on complex projects

Common Use Cases

  • Tutorials: Combine step-by-step screenshots with text annotations
  • Bug Reports: Show the sequence of actions leading to an error
  • Comparisons: Place before/after screenshots side by side
  • Conversations: Stitch chat screenshots to show full context
  • Documentation: Create comprehensive visual guides

Mobile Usage

ScreenStitch works great on mobile devices too!

  • Touch Controls: Tap and drag to move screenshots
  • Pinch to Zoom: Use two fingers to zoom in/out
  • Mobile Menu: Access all tools from the hamburger menu
  • Floating Toolbar: Quick access to common tools via the magic wand button

Troubleshooting

Images Won't Upload

  • Check that your file format is supported (PNG, JPG, JPEG, GIF, WebP)
  • Ensure files aren't corrupted
  • Try refreshing the page and uploading again

Performance Issues

  • Close other browser tabs to free up memory
  • Reduce the number of screenshots on the canvas
  • Use smaller image files when possible

Can't See My Screenshots

  • Try zooming out to see if they're outside the visible area
  • Use the reset zoom button to fit everything on screen
