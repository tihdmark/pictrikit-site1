```bash

cd dist

# 2. 部署到 Vercel
vercel --prod
```

### 部署步骤：

1. **运行部署命令**
   ```bash
   cd dist
   vercel --prod
   ```

2. **配置域名**
   - Vercel 会给你一个临时域名
   - 在 Vercel 控制台添加自定义域名: `pictrikit.com`
   - 按照提示配置 DNS 记录

3. **等待 DNS 生效**
   - 通常需要 5-30 分钟
   - 可以使用 `nslookup pictrikit.com` 检查
